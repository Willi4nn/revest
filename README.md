# Revest

Construído com **React**, **TypeScript**, **Vite**, **Tailwind CSS** e **Google Gemini API**.

## Começando

1. **Instale as dependências:**
   ```sh
   npm install
   ```

2. **Configure a chave da API:**
   
   Crie um arquivo `.env` no diretório raiz:
   ```
   VITE_GEMINI_API_KEY=sua_chave_aqui
   ```

3. **Execute o aplicativo:**
   ```sh
   npm run dev
   ```

## Recursos

* **Upload Inteligente**: Envie imagens de móveis e tecidos facilmente.
* **Processamento com IA**: Retexturização realista que preserva iluminação e geometria.
* **Download**: Salve os resultados da simulação instantaneamente.
